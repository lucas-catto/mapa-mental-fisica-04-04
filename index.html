Carga Elétrica

Q = n . e

Q => valor da carga elétrica (C)

n => quantidade de elétrons

e => carga elétrica fundamental (1,6.10^19 C)

Processos de Eletrização

Atrito

q = ± √(k * |q1| * |q2|) / r

q => carga final de cada corpo (C)

k => constante eletrostática (9 x 10^9 N m²/C²)

q1 e q2 => cargas iniciais dos corpos (C)

r => distância entre os corpos (m)

Contato

F = k * (q1 * q2) / r²

F => força eletrostática ( N)

k => constante eletrostática (aproximadamente: 8,99 x 10^9 N m²/C²)

q1 e q2 => grandezas das cargas elétricas dos corpos após a eletrização (C)

r => distância entre os centros das cargas (m)

Indução

Campo elétrico:

E = k * q / r²

Potencial elétrico:

V = k * q / r

F => força eletrostática (N)

E => campo elétrico (V/m)

V => potencial elétrico (V)

k => constante eletrostática do vácuo (aproximadamente: 8,99 x 10^9 N m²/C²)

q1 => carga do corpo indutor (C)

q2 => carga induzida no corpo induzido (C)

r => distância entre os centros das cargas (m)

Lei de Culomb

|F| = (k0.Q.q) / d^2

F => força eletrostática (N)

k0 => constante dielétrica do vácuo (N.m²/C²)

Q => carga elétrica (C)

q => carga elétrica de prova (C)

d => distância entre as cargas (m)
